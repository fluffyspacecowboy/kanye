# kanye
kanye - CLI tool which outputs a random kanye west quote.

## SPECIAL THANKS TO: https://github.com/ajzbc/kanye.rest FOR MAKING THIS POSSIBLE!!!

# How to use:
kanye

# How to build:
Either do make in root folder of the project(/kanye)
OR
do g++ src/main.cpp -o main -lcurl in the root folder of the project(/kanye)
